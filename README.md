# Pyfile Pattern Renamer

## Installation

### Cloning the repository

Use https to clone the repository:

```
git clone https://github.com/JoonasHeinonen/pyfile-pattern-renamer.git
```

### Installing the requirements

Now, move to the repository root:

```
cd pyfile-pattern-renamer
```

### Initializing the local virtual environment

Install the following package:

```
sudo apt install python3.12-venv
```

Run following to initialize the local development environment:

```
python3 -m venv .venv
```

Activate the local virtual environment:

```
source .venv/bin/activate
```

### Installing the requirements.txt

Run:

```
python3 -m pip install -r requirements.txt
```

### Initing the environment variables & running the app:

Copy the contents of `.env.example` to `.env` by running the following:

```
cp .env.example .env
```

You may now run the PyGame application by simply running `python app.py`.
